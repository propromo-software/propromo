---
title: Erstellen eines Jira-Monitors
description: Wie man einen Jira (Atlassian) Software-Projekt-Monitor erstellt
---

import { Steps } from "@astrojs/starlight/components";
import AsyncIframe from '@components/AsyncIframe.astro';

<Steps>

1. Erstelle einen neuen Jira API-Token

    <AsyncIframe>
      <iframe
        src="https://scribehow.com/embed/Creating_an_Atlassian_personal_access_token__mxLVPs4XTXmgZKomZhKFEw"
        style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;"
        allowfullscreen
        frameborder="0"
      ></iframe>
    </AsyncIframe>

2. Kopiere eine Projekt-URL von `https://<your-organization-name>.atlassian.net/jira/your-work`.

3. Besuche [create-monitor](https://propromo.duckdns.org/create-monitor) und wähle ein Projekt aus.

4. Kopiere die URL, zu der du weitergeleitet wirst.  
   Der Pfad sollte so aussehen: `/jira/software/projects/<your-project-name>/boards/1`.  
   Füge diese zusammen mit deinem generierten Token in die Eingabefelder ein.  
   _(Die Board-Nummer spielt keine Rolle)_

</Steps>
